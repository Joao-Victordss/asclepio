Classificador de
Mastite Bovina

Sistema em Python para classificar o risco de mastite em vacas leiteiras usando dados numéricos de sensores do úbere e temperatura, entregando uma resposta clara: "Mastite" ou "Saudável", com as probabilidades de cada classe. No conjunto de teste, o modelo atingiu 99,5% de acurácia.

O projeto

O que é este sistema?

Tecnologia aplicada à bovinocultura leiteira

A mastite é uma inflamação da glândula mamária e uma das doenças mais frequentes e onerosas na bovinocultura leiteira. Ela causa redução na produção de leite, descarte de leite contaminado, custos elevados com tratamento veterinário e, em casos severos, pode levar ao descarte do animal.

Este sistema utiliza dados numéricos coletados por sensores IoT posicionados no úbere, junto com a temperatura corporal, para, por meio de um modelo de Machine Learning (Aprendizado de Máquina), apoiar a identificação precoce do risco de mastite.

Na prática

Fluxo de uso da aplicação

O que acontece quando o usuário utiliza o classificador

1

Escolha do modo de uso

O usuário decide se quer usar um CSV com vários animais ou preencher manualmente os dados de cada registro diretamente na interface.

2

Envio das informações

Os dados dos sensores, do período pós-parto e da temperatura corporal são enviados para a aplicação para iniciar a análise.

3

Checagem do formato

A interface e a API verificam se o conteúdo está no formato esperado, com colunas obrigatórias e valores válidos para processamento.

4

Preparação para inferência

Depois da validação, os dados são organizados no padrão esperado pelo modelo para que a classificação seja executada corretamente.

5

Classificação pelo modelo

A API FastAPI utiliza o modelo Random Forest treinado para calcular a classe prevista e as probabilidades de mastite e de animal saudável.

6

Resultado e exportação

O sistema mostra o relatório paginado na tela, resume os totais encontrados e permite exportar os resultados em CSV.

Desenvolvimento

Como o modelo foi construído

Etapas usadas no desenvolvimento técnico do classificador

1

Definição das variáveis

Foram selecionadas as variáveis ligadas aos sensores do úbere, ao tempo pós-parto e à temperatura corporal para compor a entrada do modelo.

2

Preparação da base

A base original foi tratada para padronizar nomes de colunas, converter valores para formato numérico e remover registros incompletos ou inconsistentes.

3

Balanceamento das classes

Como problemas reais podem ter mais animais saudáveis do que com mastite, foi aplicado oversampling para reduzir viés no treinamento.

4

Separação entre treino e teste

A base foi dividida em subconjuntos para treinamento e teste, permitindo avaliar o desempenho do modelo em dados que não foram usados no aprendizado.

5

Treinamento e avaliação

O Random Forest foi treinado e depois avaliado com métricas como acurácia, matriz de confusão e análise das classes previstas.

6

Empacotamento para produção

O modelo final foi salvo em arquivo .pkl.gz para ser carregado pela API FastAPI e utilizado diretamente na interface web.

Limitações e aviso importante

Este sistema é um apoio computacional, não um diagnóstico definitivo. Ele não substitui a avaliação clínica e a conduta de um médico veterinário. O modelo atual é propositalmente objetivo e direto, usando Random Forest (Floresta Aleatória), e pode ser evoluído futuramente com comparações com outros algoritmos e técnicas de explicabilidade (formas de entender por que o modelo tomou determinada decisão).
